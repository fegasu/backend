Sintaxis de servicios de la API
METODO GET

Para solicitar todos los usuarios /usua/s/0
Para obtener datos de un usuario /usua/s/2

METODO POST

Para insertar un nuevo usuario:
datos={ "nombre":"Jose",
"apellido":"Fegasu",
"email":"jgalindos@sena.edu.co"
}
EjecutaRest('http://127.0.0.1:5000/usua/i','POST',datos))

METODO PUT

Para actualizar un usuario:
datos={
"nombre":"Jose",
"apellido":"Fegasu",
"email":"jgalindos@sena.edu.co"
}
EjecutaRest('http://127.0.0.1:5000/usua/i/8','PUT',datos))

METODO DELETE

Para borrar un registro:
EjecutaRest('http://127.0.0.1:5000/usua/i/8','DELETE',None))
